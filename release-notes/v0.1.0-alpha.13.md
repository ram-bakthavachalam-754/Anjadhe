# Anjadhe v0.1.0-alpha.13

## Added

- Actions filters now combine: pick a date window (Today, Tomorrow, This
  Week, This Month, Later — or the new **All time**) and a focus area at
  the same time, and the list narrows to both. Pills above the list show
  the active filters; the × on a pill clears just that one. A new
  **Dates** header labels the date filters like Focus Areas.
- A **No focus area** view in the Actions nav shows every task filed
  under nothing — the tasks that were invisible in every focus-area view.
- **Completed tasks show under every filter** in a collapsed "N done"
  section — Tomorrow, This Week, This Month, Later, All time, focus
  areas, and No focus area (Today already had it).
- Task rows have a **⋯ menu** (also on right-click): Open, Set date,
  Rename, Delete — quick edits without leaving the list. Same menu the
  Plan task lists already had.
- Quick-add takes its date from the page you're on: adding a task on the
  Tomorrow page schedules it for tomorrow, on Later it lands in the
  undated backlog. A date typed in the text still wins.
- Portfolio: a **Holding** transaction type to enter existing positions
  directly, and a more compact transaction editor.

## Changed

- **Task edits save automatically** — the Save button is gone from
  existing tasks; only creating a new task asks you to save. Leaving the
  task page (Back to list, Esc, or any nav click) never discards typed
  changes.
- The Actions and Plan pages drop their breadcrumbs and big headings —
  the selection pills and the left nav say what you're looking at, once.
  The task detail gets a "Back to list" link and closes on Esc.
- The old standalone Tasks and Goals pages are retired; every path into
  them (calendar events, email insights, goal links in Notes/Bookmarks,
  Pomodoro) now lands on the Actions hub or Plan's goal detail.
- "Open in Plan" is now "Edit Plan"; the focus-area view lost its
  redundant All/Today/This Week chips (the nav's date filters do that
  now, and they combine).
- Tighter Actions lists (more tasks per screen), wider Tasks | Plan
  tabs, compact task-detail buttons, and the "Restart & update" flow now
  asks before interrupting AI work that's still running.

## Fixed

- The assistant now finds schedule tasks reliably: telling it you did
  something searches your schedule (any date) instead of trusting a
  truncated list — long schedules no longer hide tasks from it — and
  when a task covers several things and you finished only one, it asks
  about the rest instead of marking the whole task complete.
- Clicking the already-active date filter while a task was open silently
  cleared the filter and dumped you on the wrong list — it now just
  takes you back.
- Background AI classifiers no longer spin in retry loops (model
  thinking disabled, decode capped).
