# Anjadhe v0.1.0-alpha.14

## Added

- **Web search is built in.** The assistant can search the web out of the
  box — no account, no API key, nothing to configure. Searches route
  through Anjadhe Connect, a small relay that forwards the query — and
  only the query — to a search provider: it keeps a count of searches,
  never what you searched, your Mac identifies itself with a random token
  rather than an account, and the provider sees queries arriving from the
  relay, not from you. 300 searches a month included; the relay's source
  code is public so all of this can be checked. Your own Tavily or Brave
  key still works exactly as before (Settings › AI Assistant › Web
  Search) and keeps Anjadhe out of the loop entirely.
- **Prompts get their own page**, sectioned by scheduled vs. on demand.
  Prompt notes no longer clutter Notes, and the Notes sidebar drops the
  Prompt Feed filter.
- **Email: per-account scoping.** The sidebar's Accounts switcher narrows
  the Email view to one account when you have several connected.
- Actions: the group labels in the left nav now act as filters too.

## Changed

- The assistant handles multi-step research more reliably: big fan-out
  questions run in task mode, real results carry between steps, and it no
  longer answers with a promise to do the work later.
- Setup and Settings copy updated for built-in search (the web-search
  help article was rewritten around it).
