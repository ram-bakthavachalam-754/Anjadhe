# Anjadhe v0.1.0-alpha.2 — the AI assistant now knows you

Second alpha. Two headline changes: when you open a chat with the AI
assistant, it already knows your focus areas, active goals, today's
schedule, what's overdue, the unread action items sitting in your
inbox, and your latest journal entry — no more "let me check…"
before the first response. And from this release forward, new
versions will auto-update.

## What's new since alpha.1

- **Daily briefing for the AI assistant.** Every conversation starts
  with a snapshot of your focus areas, in-progress goals, today's
  tasks, real overdue items, unread email action items, and your
  most recent journal entry. The assistant uses this as starting
  context, so asking "what should I focus on today?" gets an
  immediately specific answer instead of a generic tool hunt.

- **Human-friendly date formatting.** Overdue tasks, journal
  entries, and similar references appear as "yesterday", "3 days
  ago", "April 2" — never raw ISO strings.

- **Auto-updates.** New versions download in the background and
  nudge you to install with a one-click restart. You can also
  trigger a manual check from **Anjadhe → Check for Updates…** in
  the menu bar.

  ⚠️ **One-time manual upgrade for alpha.1 users.** If you
  installed alpha.1, you'll need to install alpha.2 manually this
  one time by downloading it from below and replacing the app in
  your Applications folder. From alpha.2 onward, future releases
  will come through auto-update automatically. This is a
  fundamental limitation of bolting auto-update onto a build that
  didn't have it — there's no way around it for the first hop.

- **Refreshed About view.** The in-app About screen now uses a
  hero-style layout matching the website, with the new positioning:
  a privacy-first productivity app with a built-in local AI
  assistant that lives with your data.

- **Small fixes.** Tasks with empty titles in the overdue list no
  longer confuse the assistant. Budget app hidden from marketing
  surfaces until it's wired into the AI assistant.

## Install

Download `Anjadhe.dmg` below, open the DMG, drag Anjadhe to your
Applications folder, launch from Applications. Universal binary —
runs natively on both Apple Silicon and Intel.

Upgrading from alpha.1? Replace the app in Applications. Your
data, conversations, and settings are preserved.

## Known limitations

- macOS only. Tested on macOS Tahoe; earlier versions are untested.
- Local AI requires Ollama with a Gemma 4 model (or another
  open-weight model of similar capability). The app prompts you on
  first run if Ollama isn't installed. You'll want a Mac with
  enough unified memory to run it comfortably — response speed
  scales with your hardware.
- This is still an early alpha. Please file issues in the
  [issues tab](https://github.com/ram-bakthavachalam-754/anjadhe-releases/issues)
  of this repo.

## Privacy

Your data stays on your device. No telemetry, no tracking, no
cloud servers. Personal data sent to the optional Claude
integration is automatically anonymized before leaving your
machine. See [anjadhe.com](https://anjadhe.com) for details.

## A note about the auto-attached "Source code" archives

GitHub automatically attaches "Source code (zip)" and "Source code
(tar.gz)" links to every release, and there's no way to hide them.
On this release those archives contain **only this repo's
`.gitignore`** — the actual Anjadhe application source lives in a
separate private repository. See the
[repository README](https://github.com/ram-bakthavachalam-754/anjadhe-releases#about-the-auto-attached-source-code-archives)
for more.
